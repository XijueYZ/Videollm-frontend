# VideoLLM Chat

基于 Vue.js 的视频聊天应用，支持实时视频流传输和 AI 对话。

## 功能特性

- 🎥 实时视频流传输（摄像头/屏幕录制）
- 💬 WebSocket 实时通信
- 🤖 AI 对话集成
- 📱 响应式设计

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build
```

## 技术栈

- Vue 3 + Composition API
- TypeScript
- Element Plus
- Pinia
- WebSocket
- Vite

## 开发

确保后端服务运行在 `localhost:5000`
